# Event Organisers Website

A modern, responsive website for event planning and organization services built with HTML, CSS, and JavaScript.

## 🎉 About

This is a professional event planning website featuring a clean, modern design with interactive elements. The site showcases various event services including wedding planning, corporate events, birthday parties, concerts, and more.

## 🌐 Live Demo

Check out the live version: [https://itsiamdev.github.io/Event-Organisers/](https://itsiamdev.github.io/Event-Organisers/)

## ✨ Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Interactive Navigation**: Smooth scrolling navigation with mobile hamburger menu
- **Image Gallery**: Beautiful event gallery with hover effects
- **Theme Toggler**: Dark/light theme switcher with color customization
- **Swiper Sliders**: Interactive image carousels for home and reviews sections
- **Service Showcase**: Comprehensive service offerings with icons
- **Pricing Plans**: Multiple pricing tiers (Basic, Prime, Luxury, Ultra)
- **Customer Reviews**: Testimonial section with customer photos
- **Contact Form**: Functional contact form for inquiries
- **Smooth Animations**: CSS animations and transitions throughout

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript (ES6+)**: Interactive functionality
- **Swiper.js**: Touch-enabled sliders and carousels
- **Font Awesome**: Icon library
- **Google Fonts**: Custom typography

## 🚀 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/event-organisers.git
   cd event-organisers
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - No server required for basic functionality

3. **For development**:
   - Use a local server for better development experience
   - Recommended: Live Server extension in VS Code

## 📱 Usage

### 🧭 Navigation
- Use the navigation menu to scroll to different sections
- Mobile users can access the hamburger menu

### 🎨 Theme Customization
- Click the gear icon to open theme options
- Choose from predefined color schemes
- Toggle between light and dark modes

### ⚡ Interactive Elements
- Swipe through home slider images
- Browse gallery with hover effects
- View customer testimonials in the review slider

## 🎨 Customization

### Colors
The theme colors can be customized in the CSS custom properties (CSS variables) in `style.css`:

```css
:root {
  --theme-color: #ccff33; /* Change this to your preferred color */
}
```

### Content
Update the content in `index.html`:
- Service descriptions
- Pricing information
- Contact details
- Gallery images

### Images
Replace image URLs in `index.html` with your own images:
- Home slider images
- Gallery images
- About section image
- Customer testimonial photos
